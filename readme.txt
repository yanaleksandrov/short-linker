== Installation ==

= Minimum Requirements =

* PHP 7.4 or greater is required
* MySQL 5.6 or greater is required

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handles the file transfer, and you won’t need to leave your web browser. To do an automatic install of Short Linker, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”

In the search field type Short Linker, then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

1. Upload dir `short-linker` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= Updating =

Automatic updates should work smoothly, but we still recommend you back up your site.

== Frequently Asked Questions ==

= Will Short Linker work with my theme? =

Yes! Short Linker will work with any theme, but may require some styling. Please see our [docs](http://codyshop.ru) for help.
